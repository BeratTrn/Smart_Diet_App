# 🍏 Smart Diet App

> A comprehensive, cross-platform smart diet tracking application built with Flutter. It calculates daily calories, manages macronutrients, and helps users track their health metrics seamlessly.

## ✨ Key Features

* **Advanced Health Metrics:** Dynamically calculates BMI (Body Mass Index) and daily caloric needs based on user-specific physical attributes.
* **Macro & Micro Tracking:** Real-time logging and management of proteins, carbohydrates, fats, and daily water intake.
* **Real-time Data Synchronization:** Powered by Firebase Cloud Firestore to ensure instant data consistency and offline-first capabilities.
* **Interactive Data Visualization:** Integrates dynamic charts and graphs to visualize weekly/monthly progress and nutritional habits.
* **Global i18n Support:** Fully localized application architecture supporting multiple languages (English, Turkish, German, French).
* **Secure Authentication:** Integrated Firebase Authentication for secure user onboarding and session management.

## 🛠️ Technical Architecture & Stack

* **Frontend Framework:** Flutter (Dart)
* **Backend as a Service (BaaS):** Firebase (Authentication, Cloud Firestore)
* **State Management:** Provider
* **Design Pattern:** MVVM (Model-View-ViewModel) for a scalable and maintainable codebase.
* **Asynchronous Programming:** Optimized async data fetching to prevent UI thread blocking.
